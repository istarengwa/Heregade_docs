# NpcTale - Placeholders

Permission requise: `placeholdertale.use`

## Placeholders
- `%npctale_npc_count%` - Nombre total de NPC.
- `%npctale_selected_id%` - Id du NPC selectionne.
- `%npctale_selected_role%` - Nom du role du NPC selectionne.
- `%npctale_selected_model%` - Id du modele du NPC selectionne.
- `%npctale_selected_world%` - Nom du monde du NPC selectionne.
- `%npctale_selected_created_at%` - Date de creation du NPC selectionne (epoch ms).
- `%npctale_npc_count_world<world=spawn>%` - Nombre de NPC dans le monde.
