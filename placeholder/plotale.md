# Plotale - Placeholders

Permission requise: `placeholdertale.use`

## Placeholders
- `%plotale_plot_count%` - Nombre de plots du joueur.
- `%plotale_plot_limit%` - Limite de plots du joueur.
- `%plotale_plot_limit_left%` - Plots restants.
- `%plotale_plot_list%` - Liste des ids de plots separee par des virgules.
- `%plotale_current_plot%` - Id du plot actuel.
- `%plotale_current_plot_owner%` - Id du proprietaire du plot actuel.
- `%plotale_current_plot_alias%` - Alias du plot actuel.
- `%plotale_current_plot_members%` - Nombre de membres du plot actuel.
- `%plotale_current_plot_denied%` - Nombre de joueurs refuses sur le plot actuel.
- `%plotale_current_plot_is_owner%` - Vrai si le joueur est proprietaire du plot actuel.
- `%plotale_current_plot_is_member%` - Vrai si le joueur est membre du plot actuel.
- `%plotale_current_plot_can_build%` - Vrai si le joueur peut construire sur le plot actuel.
- `%plotale_current_plot_can_use_containers%` - Vrai si le joueur peut utiliser les conteneurs sur le plot actuel.
- `%plotale_current_plot_is_admin%` - Vrai si le plot actuel est un plot admin.
- `%plotale_current_plot_merged_count%` - Nombre de plots fusionnes.
- `%plotale_current_plot_created_at%` - Date de creation du plot actuel (epoch ms).
- `%plotale_current_plot_last_activity%` - Derniere activite du plot actuel (epoch ms).
- `%plotale_current_plot_home_x%` - Coordonnee X du home du plot actuel.
- `%plotale_current_plot_home_y%` - Coordonnee Y du home du plot actuel.
- `%plotale_current_plot_home_z%` - Coordonnee Z du home du plot actuel.
- `%plotale_current_plot_pvp%` - PVP actif sur le plot actuel.
- `%plotale_current_plot_mob_spawning%` - Apparition des mobs active sur le plot actuel.
- `%plotale_current_plot_fly%` - Fly actif sur le plot actuel.
- `%plotale_current_plot_weather%` - Meteo active sur le plot actuel.
- `%plotale_current_plot_fixed_time%` - Temps fixe du plot actuel.
- `%plotale_current_plot_game_mode%` - Mode de jeu force du plot actuel.
- `%plotale_current_plot_entry_message%` - Message d'entree du plot actuel.
- `%plotale_current_plot_exit_message%` - Message de sortie du plot actuel.
- `%plotale_is_plot_world%` - Vrai si le joueur est dans un monde plot.
- `%plotale_total_plots%` - Nombre total de plots.
