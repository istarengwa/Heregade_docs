# Rankuptale - Placeholders

Permission requise: `placeholdertale.use`

## Placeholders
- `%rankuptale_rank%` - Id du rang actuel.
- `%rankuptale_rank_display%` - Nom affiche du rang actuel.
- `%rankuptale_next_rank%` - Id du prochain rang.
- `%rankuptale_next_rank_display%` - Nom affiche du prochain rang.
- `%rankuptale_next_rank_ready%` - Vrai si le prochain rang est pret.
- `%rankuptale_next_rank_cost_money%` - Cout en argent du prochain rang.
- `%rankuptale_next_rank_cost_exp%` - Cout en exp du prochain rang.
- `%rankuptale_next_rank_votes%` - Nombre de votes requis pour le prochain rang.
- `%rankuptale_next_rank_requirements_total%` - Nombre total de prerequis pour le prochain rang.
- `%rankuptale_next_rank_requirements_supported%` - Nombre de prerequis supportes.
- `%rankuptale_next_rank_requirements_completed%` - Nombre de prerequis completes.
- `%rankuptale_next_rank_requirements_remaining%` - Nombre de prerequis restants.
- `%rankuptale_next_rank_requirements_unsupported%` - Nombre de prerequis non supportes.
- `%rankuptale_next_rank_requirements_progress%` - Pourcentage d'avancement des prerequis.
