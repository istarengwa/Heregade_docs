# Essentiale - Commandes joueurs

Commandes utilisables par les joueurs.

## Commandes
- `/home [nom]` - Se teleporter vers un home (ou le home par defaut).
- `/home list` - Lister ses homes.
- `/home help` - Afficher l'aide des homes.
- `/homes` - Alias pour lister ses homes.
- `/sethome <nom>` - Enregistrer sa position comme home.
- `/delhome <nom>` - Supprimer un home.
- `/setdefaulthome <nom>` - Definir le home par defaut.
- `/warp <nom>` - Se teleporter vers un warp.
- `/warp list` - Lister les warps disponibles.
- `/warp help` - Afficher l'aide des warps.
- `/warps` - Alias pour lister les warps disponibles.
- `/spawn` - Se teleporter au spawn.
- `/rules` - Afficher les regles du serveur.
- `/tpa <joueur>` - Demander a se teleporter vers un joueur.
- `/tpa accept` - Accepter la derniere demande de teleportation.
- `/tpa deny` - Refuser la derniere demande de teleportation.
- `/tpaccept` - Alias pour accepter une demande de teleportation.
- `/tpdeny` - Alias pour refuser une demande de teleportation.
- `/tpahere <joueur>` - Demander a un joueur de se teleporter vers vous.
- `/rtp` - Teleportation aleatoire dans le monde courant.
- `/back` - Retourner a la position precedente.
- `/msg <joueur> <message>` - Envoyer un message prive.
- `/reply <message>` - Repondre au dernier message (alias: `/r`).
- `/motd` - Afficher le message du jour.
- `/kit` - Ouvrir le menu des kits.
- `/list` - Lister les joueurs connectes (alias: `/online`, `/who`).
- `/discord` - Afficher les informations Discord.
- `/top` - Se teleporter au point le plus haut du chunk courant.
