# Plotale - Commandes joueurs

Commandes utilisables par les joueurs.

## Commandes
- `/plot claim` - Reclamer le plot ou vous etes.
- `/plot auto` - Reclamer automatiquement un plot libre et s'y teleporter.
- `/plot home [index]` - Se teleporter a l'un de vos plots (index en base 1).
- `/plot visit <joueur> [index]` - Visiter le plot d'un autre joueur.
- `/plot info` - Afficher les infos du plot actuel.
- `/plot list` - Lister vos plots reclames.
- `/plot add <joueur>` - Ajouter un membre a votre plot.
- `/plot remove <joueur>` - Retirer un membre de votre plot.
- `/plot deny <joueur>` - Interdire un joueur sur votre plot.
- `/plot undeny <joueur>` - Reautoriser un joueur exclu de votre plot.
- `/plot limit` - Activer ou desactiver l'affichage des limites du plot.
- `/plot flag list` - Lister les flags disponibles (et prix/durees si configures).
- `/plot flag <flag> <true|false>` - Activer ou desactiver un flag sur le plot.
- `/plot tpback` - Retour a la position precedente avant un /plot home ou /plot auto.
- `/plot help` - Afficher l'aide des commandes plot.
