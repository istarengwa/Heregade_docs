# Picturtale - Commandes staff

## Partie joueurs
- Aucune commande joueur exposee.

## Partie admin (commandes + permissions)
- `/pict menu` - Ouvrir le menu Picturtale (dossiers + images + placeholders). Permission: `picturtale.admin.menu`.
- `/pict reload` - Recharger la config et resynchroniser les images. Permission: `picturtale.reload`.

## Permissions supplementaires
- `picturtale.admin.*` - Acces complet aux menus admin Picturtale.
