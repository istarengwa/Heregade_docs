# Picturtale - Wiki

## Fonctionnalites
- Placeholders d'images pour UI (retourne un chemin Common/Pictures).
- Resolution par dossier, chemin ou image directe.
- Indexation auto des dossiers/images.
- Sync runtime optionnel vers l'asset pack.
- Menu admin pour parcourir les dossiers/images et copier les placeholders.
- Previsualisation des images dans le menu via l'API Picturtale.
- Reload de la config + resync des images.

## Permissions
- `picturtale.admin.menu` - Ouvrir le menu admin.
- `picturtale.reload` - Recharger Picturtale.
- `picturtale.admin.*` - Acces complet aux menus admin.
