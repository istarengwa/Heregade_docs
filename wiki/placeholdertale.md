# Placeholdertale - Wiki

## Fonctionnalites
- API de placeholders (%placeholder% + params <key=value>).
- Support de scopes (global, chat, message).
- Placeholders par defaut Hytale + Heregade (Essentiale, Vaultial, Jobstale, NpcTale, Plotale, Rankuptale).
- Commande /placeholder pour tester.
- Auto-update des placeholders (intervalle configurable).
- Generation auto de docs placeholders au demarrage.
- Permission placeholdertale.use pour activer la resolution.
